загружаешь репу 

```bash
git clone https://github.com/dfyzicka/ngWG.git
```
переходишь в неё 
```bash
cd ngWG/
```
делаешь файлы иполняемыми 
```bash
sudo chmod +x *.sh
```

```bash
./sshEdit.sh
```

```bash
./install.sh
```
